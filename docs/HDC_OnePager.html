Informed Bit-Position Pruning for Hyperdimensional Computing on FPGA

A discriminability-driven Pareto study with cross-subject transferability — on Zynq, for EMG classification
Primary venue
DATE 2027
Submission
~Sept 2026

Background. Hyperdimensional Computing (HDC) represents information as wide binary vectors on which all operations are bit-parallel and noise-tolerant — a strong match for FPGA edge inference. I already have a verified 1024-bit bind + permute SystemVerilog core with an AXI-Lite wrapper (71/71 ModelSim tests passing). This proposal turns it into a streaming HDC classifier and adds three layered research contributions absent from prior FPGA-HDC work: (i) an accuracy/energy Pareto across dimension, bundle precision, and bit-pruning; (ii) a direct informed-vs-random bit-pruning comparison establishing that bit-position matters, not just bit-count; (iii) a cross-subject mask-transferability study on EMG.

What I will build

  • Six new RTL modules: item_memory, bundle_unit, pruning_mask (Hook A), popcount_am, encoder_top, hdc_stream_wrapper; all parameterised on D.
  • AXI-Stream + DMA replacing AXI-Lite to unlock streaming throughput.
  • Bare-metal C software + a Python training pipeline that emits a per-bit pruning mask offline.
  • Bit-exact Python golden reference + 10 k randomised co-sim vectors + dataset replay.

How I will evaluate it

  • Primary: UCI EMG Hand-Gesture (4 ch, 5 classes, 36 subjects) — canonical HDC benchmark.
  • Secondary: MIT-BIH ECG arrhythmia (5 AAMI classes).
  • Baselines: ARM-only HDC, the existing AXI-Lite path, a tiny int8 MLP, prior FPGA HDC.
  • Metrics: accuracy, latency, energy/inference (shunt + INA219), LUT/FF/BRAM, fmax.

Novelty / contribution

  • Hook A — three-axis Pareto: D × CNT_W × bit-pruning ratio, jointly measured on real Zynq hardware. Bit-position pruning uses per-bit Fisher discriminability and a tiny pruning_mask module gating the AM popcount.
  • Twist 1 — informed vs. random: at each pruning ratio, the discriminability-driven mask is compared head-to-head with a random mask of identical density. Expected headline: informed beats random by ≥ 5 pp at iso-budget — converts the "is this just feature selection?" objection into the strongest published claim.
  • Twist 2 — cross-subject transfer: mask trained on 18 EMG subjects, evaluated on 18 held-out subjects. Either result is publishable (universal vs. per-subject).
